4.1. Celestron alignment stars#

There are about 250 named stars in the hand control database, only 82 stars brighter than magnitude 2.5 can be used for alignment.

Note

The Celestron database uses some nonstandard names and spelling for the names of stars.

Stars marked with an asterix are below -40 degrees declination and are not suitable for alignment at BFRMO.
